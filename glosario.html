Glosario

HTML (HyperText Markup Language):

Es un lenguaje de marcado utilizado para estructurar el contenido de una página web. Permite definir la jerarquía y la organización de los elementos en una página.

CSS (Cascading Style Sheets):

Un lenguaje de estilo utilizado para describir la presentación de un documento HTML. CSS define cómo se deben mostrar los elementos, controlando aspectos como el color, el diseño y la tipografía.

Elemento:

En HTML, un elemento es una parte fundamental de la estructura de la página, representado por etiquetas. Cada elemento tiene un propósito específico y puede contener o estar contenido por otros elementos.

Etiqueta:

Un fragmento de código HTML que define o marca un elemento en la página. Las etiquetas se utilizan para estructurar y dar formato al contenido.

Atributo:

Propiedades adicionales que se pueden añadir a las etiquetas HTML para proporcionar información adicional sobre un elemento. Los atributos pueden afectar la apariencia o el comportamiento del elemento.

Selector:

En CSS, un selector es un patrón que se utiliza para seleccionar y estilizar un conjunto específico de elementos HTML. Puede ser un elemento, una clase o un identificador.

Clase:

En HTML y CSS, una clase es un atributo que se utiliza para agrupar elementos similares y aplicar estilos comunes a ellos. Puede ser reutilizada en varios elementos.

ID:

En HTML, un ID es un atributo único asignado a un elemento para identificarlo de manera única en la página. Se utiliza para aplicar estilos específicos o para referenciar el elemento en scripts.

Modelo de Caja:

En CSS, el modelo de caja describe cómo se representan visualmente los elementos HTML. Incluye propiedades como el margen, el borde, el relleno y el contenido.

Flexbox:

Una técnica de diseño en CSS que facilita la creación de diseños flexibles y responsivos. Permite distribuir el espacio entre elementos de manera eficiente, especialmente útil en diseños de interfaz de usuario.

Grid:

En CSS, Grid es un sistema de diseño bidimensional que permite organizar los elementos en filas y columnas. Es especialmente útil para crear diseños complejos y estructurados.

Responsivo (Responsive):

En el contexto web, un diseño responsivo se ajusta automáticamente según el dispositivo o tamaño de pantalla. Utiliza media queries y otras técnicas para proporcionar una experiencia de usuario óptima en diferentes dispositivos.
